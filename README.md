# How to use it? 

### Getting the lyrics
Open main.py, and change the name of the artist that you want.

### Generating the lyrics
Open the notebook you want: 
  - FragmentedGenerator.ipynb: LSTMs. Feel free to change the parameters of the network to run your own tests. You can also change the seed for generation.
  - transformers.ipynb: uses the transformers. You can change the parameters of the training or the seed for the generation.
  - Fine-TuneGPT2.ipynb: It is possible to change the model you want to use (gpt2, gpt2-medium, gpt2-large, distilgpt2, ...) and the seed.
